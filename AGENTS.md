# CMM - Commit Message Generator

## Commands

```bash
bun run start    # Run the CLI
bun run setup    # Install deps and link globally
```

## Tech Stack

- Runtime: Bun
- AI: Vercel AI SDK with Google Gemini 3 Pro
- Config: ~/.config/cmm/config.json

## Guidelines

- Use Bun, not Node/npm/pnpm
- No emojis in CLI output
- Use ANSI colors for terminal styling
